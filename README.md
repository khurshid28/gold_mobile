## O'rnatish

1. Dependencies ni o'rnatish:
```bash
flutter pub get
```

2. **MUHIM: Fontlarni o'rnatish:**

Google Fonts dan yuklab oling:
- [Poppins](https://fonts.google.com/specimen/Poppins) - Download Family tugmasini bosing
- [Playfair Display](https://fonts.google.com/specimen/Playfair+Display) - Download Family tugmasini bosing

Keyin `assets/fonts/` papkasiga quyidagi fayllarni joylashtiring:
- `Poppins-Regular.ttf`

- `Poppins-SemiBold.ttf`
- `Poppins-Bold.ttf`
- `PlayfairDisplay-Regular.ttf`
- `PlayfairDisplay-Bold.ttf`

3. Ilovani ishga tushirish:
```bash
flutter run
```
